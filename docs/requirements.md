# 分组大师 - 需求文档

## 1. 成员管理

### 1.1 用户注册
- 用户通过账号名、昵称、密码注册账号
- 可选填写备用邮箱、上传头像
- 账号名规则：3-50 位，仅允许字母、数字、下划线
- 密码规则：至少 8 位
- 账号名和邮箱（如填写）全局唯一
- 注册成功后自动登录

### 1.2 用户登录
- 使用账号名 + 密码登录
- 支持持久登录：关闭浏览器后 30 天内无需重新输入（Cookie Session）
- 登录失败显示明确错误提示

### 1.3 用户登出
- 用户可主动退出登录，清除服务端 Session

### 1.4 修改密码
- 登录用户可修改密码，需验证旧密码
- 新密码至少 8 位

### 1.5 忘记密码
- 用户输入备用邮箱，系统发送重置密码链接
- 重置链接 30 分钟内有效
- 未注册邮箱不暴露用户是否存在（统一返回提示）
- 重置后销毁该用户所有 Session，需重新登录

### 1.6 个人资料
- 查看和修改昵称
- 上传头像（支持 JPG/PNG/GIF，最大 2MB）

### 1.7 路由鉴权
- 未登录访问需登录页面 → 重定向到登录页
- 已登录访问登录/注册页 → 重定向到首页
- 页面刷新后自动恢复登录状态

### 1.8 注销账号
- 用户可在设置页注销账号
- 操作前弹出确认提示，防止误操作
- 注销后删除用户数据、头像文件及关联的 session 和重置令牌

## 2. 活动管理

### 2.1 创建活动
- 登录用户在首页可创建活动
- 活动信息：标题（必填）、描述（可选）
- 标题不能为空
- 创建成功后活动归属于当前用户
- 创建成功后出现在首页活动列表中

### 2.2 活动列表
- 登录后的首页展示当前用户创建的所有活动
- 按创建时间倒序排列
- 每条显示活动标题、描述（截断显示）
